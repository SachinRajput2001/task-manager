# Task Manager — Frontend Project

https://sachinrajput2001.github.io/task-manager

A clean, modern, resume‑ready frontend project built using **HTML, CSS, and JavaScript**.

## Features
- Add tasks with priority (High/Medium/Low)
- Delete tasks
- Fully responsive UI
- Light/Dark theme toggle (saved in localStorage)
- Tasks are saved in browser LocalStorage
- Professional UI suitable for portfolio or resume

## How to Run
Just open `index.html` in your browser — no backend needed.

## Good for Resume
This project demonstrates:
- State management in frontend
- DOM manipulation
- Responsive UI
- Theme switching
- Component‑like card layout
- Clean folder structure

You can deploy it on GitHub Pages, Netlify, or Vercel.
